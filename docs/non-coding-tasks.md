# 코딩 외 작업 관리 문서

## 사용 규칙
- 이 문서는 개발 코드 변경 없이도 진행 가능한 업무를 기록한다.
- 모든 작업자는 새 항목을 추가할 때 `ID`, `상태`, `담당`, `증빙`을 채운다.
- 상태값은 `TODO`, `DOING`, `DONE`, `BLOCKED` 중 하나만 사용한다.
- 항목 완료 시 날짜를 `YYYY-MM-DD` 형식으로 기록한다.
